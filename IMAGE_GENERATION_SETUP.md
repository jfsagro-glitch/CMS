# Настройка генерации изображений через ИИ

## Описание

Система автоматически генерирует изображения оцениваемых объектов через ИИ на основе их описания. Изображения включаются в PDF отчеты об оценке.

## Поддерживаемые сервисы

### 1. OpenAI DALL-E (Рекомендуется)

**Преимущества:**
- Высокое качество изображений
- Быстрая генерация
- Надежный API

**Настройка:**
1. Получите API ключ на [platform.openai.com](https://platform.openai.com/api-keys)
2. Создайте файл `.env` в корне проекта (если его нет)
3. Добавьте строку:
   ```
   VITE_OPENAI_API_KEY=sk-your-api-key-here
   ```
4. Перезапустите dev-сервер или пересоберите проект

**Стоимость:** ~$0.04 за изображение (1024x1024)

### 2. Hugging Face Stable Diffusion (Альтернатива)

**Преимущества:**
- Бесплатный доступ (с ограничениями)
- Открытая модель

**Настройка:**
1. Зарегистрируйтесь на [huggingface.co](https://huggingface.co)
2. Создайте Access Token в настройках профиля
3. Добавьте в `.env`:
   ```
   VITE_HUGGINGFACE_API_KEY=your-token-here
   ```
4. Перезапустите dev-сервер

**Ограничения:**
- Может быть медленнее
- Может требовать ожидания при первом запросе (cold start)

## Fallback режим

Если API ключи не настроены, система автоматически использует SVG placeholder с информацией об объекте. Это позволяет системе работать без внешних зависимостей, но без реальных изображений.

## Использование

1. Заполните форму оценки объекта
2. Нажмите "Оценить объект"
3. Система автоматически сгенерирует изображение после получения оценки
4. При выгрузке PDF изображение будет включено в отчет

## Технические детали

- Сервис: `src/services/ImageGenerationService.ts`
- Промпты автоматически оптимизируются для разных типов объектов
- Изображения кэшируются в состоянии компонента
- При ошибке генерации используется fallback placeholder
